# Description CV

Projet personnel Cloud & DevOps: conception d’un pipeline CI/CD de bout en bout pour une application Flask.

Réalisations:

- création d’une application Flask légère et structurée
- mise en place de tests automatisés avec Pytest
- conteneurisation de l’application avec Docker
- automatisation CI/CD avec GitHub Actions
- build et publication automatique de l’image Docker sur Docker Hub
- préparation du projet pour un déploiement futur sur serveur Linux ou cloud

Compétences mobilisées:

- Git et GitHub
- GitHub Actions
- Docker
- Docker Hub
- CI/CD
- tests automatisés
- variables d’environnement
- gestion de secrets

Résultat:

mise en place d’une base professionnelle, automatisée et reproductible pour industrialiser le cycle de vie applicatif.
